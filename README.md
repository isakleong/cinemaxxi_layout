# cinemaxxi_layout

Aplikasi Cinema XXI Layout

## Getting Started

Saat menjalankan Aplikasi Cinema XXI Layout untuk pertama kalinya, sistem akan meminta untuk memasukkan data kursi yang tersedia di Bioskop.
Data yang dimasukkan adalah Label Kursi dan Jumlah Kursi (maksimal 20 kursi).
Sebagai contoh inputan, Label Kursi : A, Jumlah Kursi : 20.

Jika sudah berhasil memasukkan data label kursi beserta jumlahnya, maka bisa mengakses 4 menu utama dari Aplikasi Cinema XXI Layout, yaitu:
- Pemesanan Kursi
- Pembatalan Kursi
- Laporan Denah Kursi
- Laporan Transaksi Kursi
